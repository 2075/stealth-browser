This machine has internet access.

The Stealth Browser supports using a TOR Proxy to access the Darknet. Please note that the exchange with local peers won't work when the TOR Proxy is used.

This machine will use TOR Proxy when available.

Peers

The Stealth Browser supports Peer-to-Peer transfer of already archived and cached resources

The Mode for each Peer influences what types of resources are loaded. If you're unsure what Modes do, take a look at the stealth:welcome page.

Hosts

The Stealth Browser has its own DNS resolution cache in order to prevent tracking capabilities of Internet Service Providers when being online via 2G/3G/4G.

If a domain has an AAAA entry and supports IPv6, Stealth Browser will connect to it via IPv6.

Sites

The Stealth Browser has a Stealth Mode which will influence on what is loaded by default. If a Domain has a Site-specific Mode set, it is incrementally added to the default Stealth Mode.

That means if you do not trust a Domain anymore, you need to remove its Site-specific Mode setting. On the other hand if you trust a Domain, you can override its settings by using this list here.
